# myKisah 🌱

**myKisah** adalah web application berbasis **Blazor**, **C#**, dan **.NET** yang membantu mahasiswa membangun kebiasaan positif melalui **journaling**, **habit tracking**, serta **dukungan sosial** dari buddy dan character companion.

Aplikasi ini berfokus pada **konsistensi pengguna** dalam menjalankan aktivitas harian seperti belajar, ibadah, dan self-improvement dengan pendekatan reflektif dan suportif.

---

## 📌 Latar Belakang Masalah

Banyak mahasiswa gagal menjaga konsistensi kebiasaan karena:
- Tidak memiliki sistem tracking yang terstruktur
- Tidak adanya support system atau accountability partner

Berdasarkan data penggunaan aplikasi seperti **Duolingo**, fitur **reminder** dan **companion** terbukti mampu meningkatkan **retensi harian pengguna**. myKisah mengadopsi pendekatan serupa dalam konteks pengembangan kebiasaan positif.

---

## 🎯 Solusi yang Ditawarkan

myKisah menyediakan fitur utama berikut:

- 📔 **Journaling harian** sebagai sarana refleksi diri
- ✅ **Habit tracking** untuk memonitor progres kebiasaan
- 🤝 **Buddy system** untuk saling mengingatkan dan memberi dukungan
- 🎭 **Character companion** untuk meningkatkan engagement pengguna

---

## 👥 Target User

- Mahasiswa usia **18–25 tahun**
- Pengguna yang ingin membangun **habit positif**
- Komunitas kampus atau **circle kecil**

---

## 💎 Value Utama

- Meningkatkan **konsistensi**
- Memberikan **dukungan sosial**
- Membuat proses tracking lebih **engaging dan menyenangkan**

---

## 🔄 Alur Aplikasi

1. **User Registration & Login**  
   User membuat akun dan login ke sistem

2. **Setup Awal**
   - Memilih habit yang ingin dijalankan
   - Memilih character companion

3. **Daily Activity**
   - Mengisi journal harian
   - Mengisi checklist status habit

4. **Buddy Interaction**
   - Menambahkan buddy
   - Melihat journal dan progres habit buddy

5. **Companion Feedback**
   - Respon character terhadap journal
   - Feedback character terhadap progres habit

6. **Progress Monitoring**
   - Melihat riwayat journal
